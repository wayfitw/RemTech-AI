# Ремтехника — Генератор КП

> **Статус: референс для портирования, не прод.**
>
> Каталог содержит самостоятельный прототип на Node.js. Он служит донором логики для переноса в основную платформу RemTech-AI (Python-бэкенд) и не предназначен для продакшн-развёртывания. Причины: нет аутентификации, запрос к провайдеру модели идёт напрямую в обход шлюза моделей и контура 152-ФЗ, база данных отсутствует.
>
> Перенос ведётся в задаче TASK-0507 (issue #45). Разбор и план переноса: `Remtechnika-AI-Docs/04_Аналитика_и_решения/Решение_kp-generator_Модуль1_2026-07-18.md`, скелет целевой функции приложен к issue #45.

Веб-приложение для автоматического формирования коммерческих предложений.

## Быстрый старт

### 1. Установка зависимостей

```bash
cd remtechnika-kp
npm install
```

### 2. Настройка API ключа

Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Откройте `.env` и вставьте свой ключ:

```
ANTHROPIC_API_KEY=sk-ant-...
PORT=3000
```

### 3. Запуск

**На Mac (локально):**
```bash
node server/index.js
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)

## Публичный доступ через ngrok

Чтобы дать доступ клиенту или коллеге:

**На Mac:**
```bash
# Установка ngrok: https://ngrok.com/download
ngrok http 3000
```

Скопируйте ссылку вида `https://xxxx.ngrok.io` и отправьте клиенту.

## Как пользоваться

1. **Шаг 1** — заполните реквизиты (проект, клиент, менеджер) и загрузите PDF/DOCX от поставщика или вставьте текст вручную
2. **Шаг 2** — проверьте и отредактируйте извлечённые данные
3. **Шаг 3** — скачайте готовый PPTX файл

## Структура проекта

```
remtechnika-kp/
├── server/
│   ├── index.js      # Express сервер и API роуты
│   ├── parser.js     # Парсинг PDF/DOCX
│   ├── ai.js         # Запросы к Claude API
│   └── generator.js  # Генерация PPTX
├── client/
│   ├── index.html
│   ├── style.css
│   └── app.js
├── uploads/          # Временные файлы (очищаются автоматически)
├── output/           # Готовые PPTX (удаляются после скачивания)
└── .env              # API ключ (не коммитить!)
```

## Модель Claude

По умолчанию используется `claude-sonnet-4-5-20251022`. Изменить можно в `server/ai.js` (строка `model:`).
